# Market Intelligence

## Purpose

This folder stores GO IRL market intelligence, competitor monitoring, and repeatable research process.

It prevents competitor ideas from becoming MVP scope without product filtering.

## Source-of-truth hierarchy

1. `docs/MARKET_POSITIONING.md` defines GO IRL positioning and MVP feature filter.
2. `docs/COMPETITOR_WATCH.md` stores known competitor categories and product signals.
3. `docs/market/CONTINUOUS_COMPETITOR_INTELLIGENCE.md` defines ongoing monitoring process.
4. `docs/market/COMPETITOR_ANALYSIS_TEMPLATE.md` defines how to analyze one competitor.

## Current beta thesis

GO IRL is a Telegram-first local meetup layer for small real-life activities in Olomouc.

It is not:

- an event calendar;
- a ticketing platform;
- a sport-only app;
- a dating product;
- a social feed;
- a club CRM.

## Market rule

Competitor features are only inputs.

Before any competitor-inspired feature enters MVP scope, ask:

> Does this make it easier for people to leave the chat and meet in real life?

If the answer is no, the feature stays future scope.

## Files

- `CONTINUOUS_COMPETITOR_INTELLIGENCE.md` - repeatable monitoring process.
- `COMPETITOR_ANALYSIS_TEMPLATE.md` - template for individual competitor analysis.

## Review cadence

- Light review: monthly.
- Full competitor refresh: quarterly.
- Immediate review: if a direct local competitor appears in Olomouc or Telegram Mini Apps.
