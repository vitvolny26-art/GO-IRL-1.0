# Sprint 3 - Trust, Verification, RLI

## Status

Draft / future trust layer.

This file preserves the Sprint 3 trust direction. It does not make reputation, RLI, or attendance confirmation current MVP scope.

## Goal

Start building GO IRL's trust layer around real attendance.

## Planned scope

- Attendance confirmation.
- Organizer participant verification.
- Participant-to-participant verification.
- RLI history and basic profile reputation.
- Achievements tied to real participation.

## Current MVP boundary

Current beta trust is focused on:

- clear event cards;
- organizer/host trust;
- participant count and capacity;
- event chat;
- Sport Coach support for beginner comfort;
- Telegram share/join loop.

## Deferred scope

Do not build before explicit approval:

- public Trust Score;
- public ratings;
- leaderboard;
- token/reward mechanics;
- attendance geolocation confirmation;
- complex reputation UI.
