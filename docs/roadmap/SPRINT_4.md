# Sprint 4 - Modules And Discovery

## Status

Draft / future capability.

This file preserves the Sprint 4 direction. It does not authorize MVP expansion beyond the current Olomouc beta scope.

## Goal

Evolve from a generic event list into a modular platform.

## Planned scope

- Sport as the first strong module.
- Module-specific cards, filters, and creation fields.
- Activities, Nature, Parties, Creative, and Learning prepared as independent modules.
- City expansion through configuration.

## Current implementation boundary

Current MVP keeps:

- Sport as the reference vertical;
- Generic fallback for non-sport activities;
- six canonical Olomouc beta categories;
- no broad multi-city catalog before beta validation.

## Current beta categories

1. Volleyball
2. Running
3. Walking
4. Coffee meetup
5. Board games
6. Language exchange

## Deferred scope

Do not start before beta evidence:

- broad lifestyle verticals;
- full city catalog;
- Friends vertical;
- Travel vertical;
- Dating vertical;
- AI recommendations.
