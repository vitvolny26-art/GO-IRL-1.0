# Sprint 1 - MVP Core

## Status

Historical execution record.

This file preserves the intended Sprint 1 scope. Current MVP scope is controlled by `ROADMAP.md`, `BACKLOG.md`, `README.md`, and `DOCS_INDEX.md`.

## Goal

Make the main user journey feel clear, fast, and useful.

## Planned scope

- Premium event cards that answer what, when, where, who, price, and join status.
- Redesigned home screen around discovery and categories.
- Activity creation in under 30 seconds.
- Join/request flow in under 15 seconds.
- Organizer edit and private request review.
- Strong empty, loading, success, and error states.

## Current relevance

Sprint 1 still maps to the closed beta loop:

```text
create event -> share -> participants join -> event chat -> people show up in real life
```

## Guardrail

Do not expand Sprint 1 into social feed, direct messages, ticketing, payments, or dating scope.
